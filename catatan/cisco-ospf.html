Konfigurasi Routing Dinamis OSPF di Cisco

Dipelajari pada: 10 Juni 2026 | Kategori: #Networking

Hari ini saya berhasil menyelesaikan praktikum OSPF (Open Shortest Path First) menggunakan Cisco Modeling Labs. OSPF adalah routing protocol dinamis yang menggunakan konsep area untuk efisiensi.

Router(config)# router ospf 1
Router(config-router)# network 192.168.1.0 0.0.0.255 area 0
Router(config-router)# network 10.10.10.0 0.0.0.255 area 0
Router(config-router)# exit

Hasil: routing antar network berjalan otomatis tanpa perlu static route.
