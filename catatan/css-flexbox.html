Eksplorasi CSS Flexbox untuk Layout Modern

Dipelajari pada: 11 Juni 2026 | Kategori: #WebDev

Flexbox sangat membantu membuat layout responsif tanpa ribet. Hari ini saya coba membuat navbar dan card grid dengan Flexbox murni.

.container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}
.card {
  flex: 1 1 200px;
}

Hasilnya rapi di semua ukuran layar.
